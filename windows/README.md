# ⌨️ Kurmancî Keyboard Layout — Windows

## بالعربية
1. اذهب إلى قسم **[Releases](../../../releases)**.
2. حمّل الملف المضغوط `krlat_us.zip`.
3. فك الضغط عن الملف، ثم شغّل **`setup.exe`** لبدء التثبيت.

يحتوي هذا المجلد أيضاً على حزم `.msi` منفصلة لكل معمارية (amd64 / i386 / ia64) والمكتبات (`.dll`) المرافقة لها، لمن يفضّل التثبيت اليدوي أو النشر عبر أدوات إدارة الأجهزة (مثل Group Policy / Intune).

## Bi Kurmancî
1. Herin beşa **[Releases](../../../releases)**.
2. Pelê `krlat_us.zip` daxînin.
3. Pelê zip vekin û **`setup.exe`** bimeşînin.

Ev peldank her wiha pakêtên `.msi` yên cuda (amd64 / i386 / ia64) û pelên `.dll` yên tevlihev vedihewîne, ji bo sazkirina destan an belavkirina bi rêya amûrên rêveberiya cîhazan (Group Policy / Intune).
