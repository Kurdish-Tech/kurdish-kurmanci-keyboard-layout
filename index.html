📥 Daxistin / تحميل / Download 📁 windows/ (amd64, i386, ia64, wow64)

Full Package installer compatible with Windows 10 & Windows 11

Bi Kurmancî

  1. Li jorê bitikînin ser bişkoja şîn a Daxistinê.
  2. Pelê krlat_us.zip daxînin ser komputera xwe.
  3. Pelê zip vekin (Extract) û herin hundirê peldankê.
  4. Pelê setup.exe du caran bitikînin û sazkirinê biqedînin.

باللغة العربية

  1. انقر على زر التحميل الأزرق البارز في أعلى الصفحة.
  2. قم بتحميل الملف المضغوط krlat_us.zip على جهازك.
  3. فك الضغط عن الملف المستلم وافتح المجلد الناتج.
  4. اضغط مرتين على ملف التثبيت setup.exe واتبع الخطوات البسيطة لإتمام العملية.

In English

  1. Click the blue Download button above.
  2. Download the krlat_us.zip file onto your machine.
  3. Extract the downloaded zip archive and open the folder.
  4. Double-click setup.exe and follow the prompt to finish.

📥 Daxistin / تحميل / Download (.keylayout) 📁 macos/ (Kurmanci.bundle)

Single-file layout — no compiling required

Not / ملاحظة: Kurdî li ser macOS bi bişkoja Option (⌥) tê çalakirin, ne AltGr. Cih heman in: [ ] \ ; '.

Bi Kurmancî

  1. Pelê Kurmanci.keylayout daxînin (bişkoja jorîn).
  2. Pelê bikişînin nav peldanka ~/Library/Keyboard Layouts/ (di Finder de: Go → Go to Folder…, ev rê binivîsin; heke peldank tune be, wê biafirînin).
  3. Derkevin û dîsa têkevin hesabê xwe (Log out / Log in).
  4. Herin System Settings → Keyboard → Input Sources → Edit… → +, lêgerînê bikin Kurmancî û zêde bikin.

باللغة العربية

  1. حمّل ملف Kurmanci.keylayout من الزر أعلاه.
  2. انسخ الملف إلى المجلد ~/Library/Keyboard Layouts/ (من Finder: Go → Go to Folder… واكتب هذا المسار، وأنشئ المجلد إن لم يكن موجوداً).
  3. سجّل الخروج ثم أعد تسجيل الدخول.
  4. اذهب إلى System Settings → Keyboard → Input Sources → Edit… → +، ابحث عن Kurmancî وأضفه.

In English

  1. Download Kurmanci.keylayout using the button above.
  2. Copy the file into ~/Library/Keyboard Layouts/ (in Finder: Go → Go to Folder…, paste this path, creating the folder if needed).
  3. Log out and log back in.
  4. Go to System Settings → Keyboard → Input Sources → Edit… → +, search for Kurmancî and add it.

📥 Daxistin / تحميل / Download (XKB symbols) 📁 linux/ (.deb / .rpm guide)

XKB symbols file — works on any X11 desktop (GNOME, KDE, XFCE…)

Not / ملاحظة: Gelek dabeşan jixwe pelekî bi navê ku tînin (Soranî). Wî pelî rasterast neguherînin — vê pelê bi navekî cuda (kumr) saz bikin, wekî li jêr hatiye destnîşankirin.

Bi Kurmancî

  1. Pelê XKB daxînin (bişkoja jorîn).
  2. Bi termînalê wê kopî bikin: 
    sudo cp ku /usr/share/X11/xkb/symbols/kumr
  3. Ji bo ceribandinê tavilê çalak bikin: 
    setxkbmap -layout kumr -variant basic
  4. Ji bo sazkirina daîmî û pakêtên .deb/.rpm, li linux/README.md binêrin.

باللغة العربية

  1. حمّل ملف XKB من الزر أعلاه.
  2. انسخه عبر الطرفية (Terminal): 
    sudo cp ku /usr/share/X11/xkb/symbols/kumr
  3. فعّله مباشرة للتجربة: 
    setxkbmap -layout kumr -variant basic
  4. للتثبيت الدائم وبناء حزم .deb/.rpm، راجع linux/README.md.

In English

  1. Download the XKB file using the button above.
  2. Copy it via terminal: 
    sudo cp ku /usr/share/X11/xkb/symbols/kumr
  3. Activate it immediately to test: 
    setxkbmap -layout kumr -variant basic
  4. For permanent setup and .deb/.rpm packages, see linux/README.md.
